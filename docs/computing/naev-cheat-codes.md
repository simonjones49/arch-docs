F2 to enter codes


Invincibility
```
player.pilot():setInvincible(true)
```

Get a local map
```
player.addOutfit( "Local System Map" )
```


Player refuel
```
player.refuel ()
```


Get more money
```
player.pay(amount)
```


Get another ship, must be on a planet
```
player.addShip( "Pirate Kestrel" ) (Bird of Prey) 
```


Change status of faction (like or hate is 100 to -100)
```
faction.modPlayerRaw("Goddard", 100)
```
